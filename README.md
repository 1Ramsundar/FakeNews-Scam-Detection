# Comparative Analysis of Classical and Transformer-Based Models for Cross-Domain Fake News Detection

A cybersecurity threat analysis dashboard that compares **Classical ML (TF-IDF + Naive Bayes)** and **Transformer-based (DistilBERT)** models for detecting fake news and scam messages.

## Features
- **Scam text analysis** — Classical & BERT models
- **Fake news detection** — Classical & BERT models
- **URL risk analysis** — heuristic domain inspection
- **Image OCR analysis** — Tesseract + ML classification
